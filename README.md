# Finatics Aquatics

Welcome to the official repository for **Finatics Aquatics** — a UK-based breeder of high-quality African cichlids including Mbuna, Peacock, and Haplochromis species.

## 🌍 Live Site
Visit: [www.finaticsaquatics.co.uk](https://www.finaticsaquatics.co.uk)

## 📦 Features
- Dynamic fish stock grid from Google Sheets
- Secure PayPal checkout
- Editable shopping cart
- Mobile-responsive layout
- SEO optimized with custom metadata, Open Graph, and Twitter cards

## 📁 Project Structure
```
/
├── index.html            # Homepage
├── cart.html             # Shopping cart and checkout
├── sitemap.xml           # Sitemap for search engines
├── robots.txt            # Crawl directives
├── /css/                 # Stylesheets
├── /js/                  # JavaScript modules (cart, stock, PayPal, etc.)
├── /images/              # Fish photos and fallback media
```

## 🛠️ Setup
This site is static and hosted via GitHub Pages (or Netlify). Simply push to main and it will auto-deploy.

## 🔍 SEO + Search Console
- Submit `sitemap.xml` via Google Search Console
- Ensure `robots.txt` is at site root and publicly visible
